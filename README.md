# IPD ITS Auto Filler Extension

Extension Google Chrome untuk membantu pengisian Kuesioner Indeks Pengajaran Dosen (IPD) dan Mata Kuliah di `akademik.its.ac.id` secara efisien, otomatis, dan dengan komentar yang sopan.

- **Deteksi Dosen/Mata Kuliah**: Secara otomatis mendeteksi dan memilih opsi pada dropdown menu (Select Option) yang tersedia.
- **Komentar Berkualitas**: Mengisi kolom saran dan komentar dengan kalimat yang panjang, sopan, formal, dan konstruktif.
- **Support Multi-Radio**: Mendukung berbagai format form radio button horizontal maupun vertikal.

## ğŸ“¦ Cara Install (Load Unpacked)

Karena ekstensi ini tidak dipublish ke Chrome Web Store (untuk privasi dan penggunaan pribadi), Anda perlu menginstalnya secara manual:

1.  **Download** atau simpan folder proyek ini di komputer Anda.
2.  Buka Google Chrome atau Microsoft Edge.
3.  Ketik `chrome://extensions` di address bar lalu tekan Enter.
4.  Aktifkan **Developer mode** (Mode pengembang) di pojok kanan atas.
5.  Klik tombol **Load unpacked** (Muat yang belum dibongkar) di pojok kiri atas.
6.  Pilih folder **`ipd-extension`** yang berisi file `manifest.json`.
7.  Extension siap digunakan! Icon akan muncul di toolbar browser.

## ğŸ“– Cara Penggunaan

1.  Login ke [akademik.its.ac.id](https://akademik.its.ac.id/).
2.  Masuk ke halaman **IPD (Isi Kuesioner)**.
3.  Buka salah satu form kuesioner mata kuliah/dosen.
4.  Klik icon **IPD Auto Filler** di toolbar Chrome.
5.  Klik tombol **"Isi Kuesioner (Sangat Baik)"**.
6.  Cek kembali isian form, lalu klik tombol **Simpan** di halaman web.
